# 验收清单（10–20 分钟）- JLPT Vocab Cards MVP

> 目的：把“30 秒冒烟”升级为可复现的 MVP 验收流程。

## A. 基础导航（~2 分钟）
- 首页加载：章节/课程列表正常渲染
- 进入 `ch01-l01`、`ch01-l02`：标题（中日）正确、页面不白屏
- 返回目录 / 进入设置页：路由正常

## B. 扑克牌背诵（~5–7 分钟）
- 长按 1 秒揭示：稳定触发；松开立即隐藏
- 左右滑动翻页：连续翻 10 张不卡顿、不乱跳
- 循环播放开关：
  - 关闭：最后一张 → 下一张不回到第一张
  - 开启：最后一张 → 下一张回到第一张
- 跳到编号：
  - 合法编号：定位正确
  - 非数字/不存在：toast 提示正确

## C. 列表/搜索（~3–5 分钟）
- 搜索：平假名/中文/英文/例句关键字能过滤到正确项
- 列表跳到编号：
  - 错误输入 toast 正常
  - 正确输入定位到卡片并切回扑克牌
- 列表行信息：显示 `word · 中文释义`，点按进入正确卡片

## D. 音频（~3–5 分钟）
- 播放/暂停/拖动进度：基础可用
- 预缓存按钮：点击后状态变化为“已缓存”（或至少无报错）
- 断网验证（可选）：预缓存后开启飞行模式，音频仍可播放
  - 若失败：记录浏览器版本与现象（可能与 Range 请求/缓存策略有关）

## E. 工程质量（~2–3 分钟，可选）
- `cd web && pnpm lint && pnpm build` 通过
- `.env.example` 存在，且说明清晰

## Done 定义（本项目）
- Claude review 仅剩低风险/纯 UX 问题
- lint/build 绿
- A-D 全部通过
